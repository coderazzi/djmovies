Set up LINUX
============
cd djmovies
sudo apt-get install mediainfo unrar imagemagick libxml2-dev libxslt-dev zlib1g-dev ffmpeg mkvtoolnix
pip3 install virtualenv
virtualenv venv
source venv/bin/activate
pip3 install -r requirements.txt 
deactivate

Set up MACOS
============
brew install mediainfo rar freetype imagemagick@6
brew install ffmpeg 
export MAGICK_HOME=/opt/homebrew/opt/imagemagick@6

=======To restore movies

cp ~/Dropbox/backups/djmovies.70.sqlite db_movies.sqlite

cp -R ~/Dropbox/backups/dj_movies_images movies/static/mov_imgs/
