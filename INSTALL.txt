Set up
======
cd djmovies
sudo apt-get install mediainfo unrar imagemagick libxml2-dev libxslt-dev zlib1g-dev ffmeg mkvtoolnix
pip3 install virtualenv
virtualenv venv
source venv/bin/activate
pip3 install -r requirements.txt 
deactivate


In Mac, it was needed:
======================
brew install mediainfo unrar freetype imagemagick@6
brew install ffmpeg --with-fdk-aac
export MAGICK_HOME=/usr/local/Cellar/imagemagick@6/6.9.9-27/


To execute
==========
./go.sh
